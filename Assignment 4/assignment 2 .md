# Assignment: Delete Records Using Subquery

## Problem Statement

The company wants to remove employees whose individual sales contribution is very low compared to the total sales generated by all employees.

Delete those employee records whose sales amount contributes less than **20%** of the total sales amount.

## Query

```sql
DELETE FROM employee_sales e1
WHERE (
    e1.sale_amount * 100.0 /
    (SELECT SUM(e2.sale_amount)
     FROM employee_sales e2)
) < 20;
```

calculates the total sales amount of all employees.

After that, the outer query calculates the percentage share of each employee using this formula:

```sql
employee sale amount * 100 / total sale amount
```

If an employee’s percentage share is less than 20%, then that employee’s record is deleted from the table.

For example, if the total sales amount is `323002.80`, then each employee’s sale amount is compared with this total. Employees whose contribution is below 20% will be removed.

This query is useful when we want to clean records based on percentage contribution instead of using a fixed amount.
